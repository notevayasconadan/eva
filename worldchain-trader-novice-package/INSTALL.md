# 🚀 Worldchain Trader - Ubuntu Installation Guide

## Quick Installation

### Option 1: One-Command Install (Recommended)
```bash
curl -fsSL https://raw.githubusercontent.com/notevayasconadan/eva/main/worldchain-trader-novice-package/install-ubuntu-one-line.sh | bash
```

### Option 2: Manual Installation
```bash
# Clone the repository
git clone https://github.com/notevayasconadan/eva.git
cd eva/worldchain-trader-novice-package

# Run the installer
./install-ubuntu.sh
```

## What Gets Installed

✅ **Node.js 18.x** - Latest LTS version

✅ **PM2 Process Manager** - Background service management  
✅ **Systemd Service** - Auto-start on boot  
✅ **Complete Configuration** - Ready-to-use setup  

## Post-Installation Steps

1. **Navigate to installation directory:**
   ```bash
   cd ~/worldchain-trader
   ```

2. **Configure your wallet:**
   ```bash
   ./setup-novice.sh
   nano .env
   ```

3. **Start the trading bot:**
   ```bash
   ./start-trader.sh
   ```
